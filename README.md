### 开启桥

#### 项目简介

市面上，常见的开启桥类型右四种，立转式，平转式，提升式和折叠式。样式如下图所示：

![1](图片\1.png)

我们小组经过思考和讨论决定采取立转式的开启桥方案，利用 arduino 单片机去驱动舵机运动，从而抬升桥面，开启桥。

#### 模型：

我们采用Fusion 360构建开启桥模型。模型如下：

![2](图片\2.png)

再用3D切割，拼接起来。实物图如下：

![3](图片\3.png)
